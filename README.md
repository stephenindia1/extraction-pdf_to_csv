# Extraction - of PDF to CSV

If you have large pdf files, from which you want to use the data
Best way is to get it extracted into a dataframe -> (to csv / or any other format or database if the data-size is big)

To extract pdf data you would require to install 2 packages, using pip
1. pip install tabula-py
2. pip install jpype1

Then run the file provided as
>> python pdf_to_csv.py
